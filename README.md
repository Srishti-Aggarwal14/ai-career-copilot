# 📌 Overview

AI Career Copilot is a complete placement preparation platform designed to help students prepare for software engineering interviews through AI-powered resume analysis, mock interviews, DSA tracking, placement analytics, job recommendations, interview scheduling and career guidance.

The project combines multiple placement preparation tools into one centralized dashboard to improve productivity and provide personalized insights.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Career-Copilot.git
```

Go inside the folder

```bash
cd AI-Career-Copilot
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm run dev
```

Open your browser

```
http://localhost:5173
```
